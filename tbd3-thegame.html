TBD 3 - The Game

Information on Project

This is a personal project that my friend and I are working on in our free time. Currently I am working on the 3D models that will be used in the game and the tool to create and save levels. For the level tool I am going to have each level be saved into a text file that will be read in and used in the game. This makes making the levels easy and fun. This is still a work in progress. For this project we are using Unity and Maya.
This game will be a sort of dungeon crawer but instead of going through a dungeon you are going through a office building where you are going to have to fight various enemies on each floor to reach the top.
All screenshots are from the Level Creator Tool. Still in early stages of development. That's why it isn't pretty looking.
