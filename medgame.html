Med Game

Information on Project

This game was made in partnership with the University of Rochester medical department. They partnered up with RIT to make a game that would help medical students study and prepare for their exams. This game focused on pregnacy and all the things that could happen during pregnacy. My job on this project was to design the user interface for the game and create all the assets needed for the game. There was no medical artist for the game so I was also incharge of drawing as accurate as possible the human body. I have included a pdf with my design notes and drawings from early on in the process. For this project Adobe Photoshop 2015 was used.
